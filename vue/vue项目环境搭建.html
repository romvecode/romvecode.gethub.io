# vue 正式项目环境搭建步骤

  1. 项目创建
// vue create +'项目名称'

以下 图片是操作步骤

  1. 项目创建完毕后配置需要的文件信息
(1)在项目文件夹下配置一个.prettierrc.js文件：内置文件内容 module.exports = {
    "eslintIntegration": true,

    "singleQuote": true,

    "semi": false
}
(2)在.eslintrc.js文件里添加两条配置：
     // 强制使用单引号
    "quotes":["error","single"],
    // 强制不使用分号结尾
    "semi":["error","never"]
(3)在src/assets/下添置需要的css,fonts,images等文件夹 里边放置需要的数据信息
  1. 需要在 github 平台创建一个远程仓库，让项目可以远程连接到仓库。具体步骤如下
(1)推送到 github 第一步：设置仓库地址
// 远程连接仓库地址
git remote add origin git@github.com:romvecode/meizu.demo.git
(2)查看远程仓库地址：git remote -v
(3)去github仓库中拉取最新代码：git pull origin master --allow-unrelated-histories
  1. 查看当前工作区中有哪些修改
git status
  1. 把代码提交到版本库
git commit -m "注释信息"
  1. 如果当前项目已经有了远程仓库地址，需要修改地址：
git remote set-url origin + 新地址
  1. 推送项目到 github
git push origin master
