# vue-cli脚手架的搭建项目

  1. 创建项目
进入到linux终端命令行，cd到指定项目文件夹下mkdir需要的脚手架项目 (vue create 项目名称)
系统默认安装了babel(注意：babel是一个编译转换工具 把ES6代码转换成ES5代码的);
以及eslint(ESLint 是一个ECMAScript/JavaScript 语法规则和代码风格的检查工具，它的目标是保证代码的一致性和避免错误。)
  1. 安装vue-router

在指定项目下 npm install --save vue-router

  1. 项目文件创建好后开始进行配置
3.1 在文件src跟目录文件下的components文件下创建一个router配置文件夹，在router文件夹下创建index.js文件。 
配置内容：(1)引入 import Vue from 'vue'
       (2)引入vuerouter 模块 import 引入模块 ES6里面的规范 require是nodejs里面使用
       import VueRouter from 'vue-router'
       (3)  让VueRouter 作为Vue的一个插件
           Vue.use(VueRouter)
       (4)配置VueRouter
       const router = new VueRouter({
           <!-- history去掉路由 hash自带# -->
           mode: "history",
           routes:[{
                {
     // 当访问的是/vip这个路径
     path: '/vip',
     // 渲染对应的组件
     component: Vip,
     children: [
       {
         path: 'one',
         component: One,
         // 路由命名
         name: 'one'
       },
       {
         path: 'two',
         component: Two
       },
       {
         path: 'three',
         component: Three
       }
           }]
       })
       (5)最后导出内容
       export default router

       3.2 在默认main.js文件下需要配置添加
       <!-- main.js 这个文件叫做入口文件 当项目启动首先访问这个文件 -->
       import Vue from 'vue'
       引入App这个组件
       import App from './App.vue'
       // 引入router模块
       import router from './router/'
       关于产品信息的提示
       Vue.config.productionTip = false

       new Vue({
       router,
       render: h => h(App),
       }).$mount('#app')
